# Spendlens: Expense Tracker PWA

Spendlens is a simple, lightweight Progressive Web App (PWA) designed to help you track your expenses. This was built as a capstone project.

It features a clean user interface for logging transactions and provides a clear overview of your spending. Because it's a PWA, it can be "installed" on your mobile device or desktop for an app-like experience.

* **Log Expenses:** Easily add new expenses with a title and amount.
* **View History:** See a running list of all your past transactions.
* **PWA Ready:** Installable on modern browsers (desktop and mobile) for offline access and a native feel.
* **Lightweight:** Built with a simple Python backend and a clean HTML/CSS/JS frontend.
